You can choose to create a second table for the 500gpbs dataset and load into the staging table and insert in the same way
```SQL
create table cisco_baseline_500gbps (like cisco_baseline);
```

Then you can run queries like

```SQL
select * from cisco_baseline;
select time, encodingPath, node_name, 
      sum(vrf__update_messages_received), 
      sum(input_data_rate), 
      sum(input_packet_rate), 
      sum(output_data_rate), 
      sum(output_packet_rate),
      sum(output_drops) 
from 
cisco_baseline sample by 1s;
```
